# Getting Started

## Step 1: Intall node_modules packages

```sh
# Using npm
npm install
```

## Step 2: Build and run your app

### iOS
Install pods before running

```sh
npx pod-install
```

In case npx pod-install fails, run:

```sh
bundle install
```

```sh
npx react-native run-ios
```

### Android

```sh
npx react-native run-android
```

# Testing Redux

```sh
npm test redux
```
